Steps

Now switch back to the cloned directory

Execute:

cd ../cloned_hello
# (You should be in the cloned hello repository)
ls

We see the changes to README are not here yet.

Fetch the Changes

Execute:

git fetch origin
cat README

Output:

$ git fetch origin
From /Users/jim/Documents/Presentations/git_immersion/auto/hello
   a4954df..9ba7cf0  master     -> origin/master
$ cat README
This is the Hello World example from the git tutorial.
(changed in original)

The changes have been copied into the repository, but have not yet been moved into the working directory. We get the changes by merging the changes in the remote branch into master.

Execute:

git merge origin/master

Pulling

The fetch/merge commands can be done in one operation with pull.

Execute:

git pull origin
